Smart & Intelligent Zebra Crossing with Automatic Barrier Control System

Overview

This project is an Arduino-based smart road safety system designed to improve pedestrian safety at zebra crossings.

Features

- Pedestrian Detection
- Automatic Barrier Control
- Traffic Signal Automation
- Buzzer Alert System
- Smart Safety Mechanism

Components Used

- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- SG90 Servo Motor
- Red LED
- Green LED
- Buzzer

Working Principle

1. Ultrasonic sensor detects pedestrian movement.
2. Red signal turns ON.
3. Green signal turns OFF.
4. Barrier closes automatically.
5. Buzzer alerts nearby vehicles.
6. After crossing, barrier opens automatically.

Applications

- Smart Cities
- School Zones
- Hospital Areas
- Busy Road Intersections

Future Scope

- AI-based Pedestrian Detection
- Camera Integration
- IoT Dashboard Monitoring
